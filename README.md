# Keep-running-n-nobody-gets-hurt
Keep running N nobody gets hurt.
3D runner game in which you have to scape from a maze full of surprises.
Implemented from scratch with OpenGL 4.5 and SFML.

Library dependencies:
	- SFML
	- GLM
	- GLEW
